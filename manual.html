📘 Manual do Simulador de Autômatos

Olá! Este manual vai te ajudar a entender o que é um autômato finito e como usar o nosso simulador. Não se preocupe se você nunca ouviu falar nisso antes – explicamos tudo passo a passo, como se fosse um jogo.

🧠 O que é um autômato finito?

Imagine uma máquina que lê uma sequência de letras (ou símbolos) e decide se ela é aceita ou não. Por exemplo, uma máquina que só aceita palavras que terminam com a letra "a".

O autômato finito é uma dessas máquinas. Ele tem:

  • Estados: são como "posições" ou "etapas" que a máquina pode estar.
  • Transições: regras que dizem: "se você está no estado X e lê a letra Y, vai para o estado Z".
  • Estado inicial: por onde a máquina começa.
  • Estados finais: se a máquina terminar em um desses estados, a palavra é aceita.

Existem dois tipos principais:

  • AFD (Autômato Finito Determinístico): para cada estado e símbolo, há no máximo uma transição. É mais simples de simular.
  • AFND (Autômato Finito Não Determinístico): pode ter várias transições para o mesmo símbolo, ou até transições espontâneas (ε). É mais flexível.

Nosso simulador reconhece automaticamente qual tipo você está criando e se adapta.

🎮 Como usar o simulador

A tela está dividida em duas partes:

  • Área do desenho (à esquerda): onde você vê os estados (círculos) e as setas (transições). Você pode arrastar os círculos para organizar melhor.
  • Painel de controle (à direita): onde você cria estados, define transições e testa palavras.

✏️ Criando seu autômato

1. Adicionar estados

Clique no botão ➕ Adicionar. Um novo círculo aparecerá. Você pode renomeá-lo clicando nele e digitando um novo nome no campo "Novo nome" e depois em "Renomear".

2. Definir o estado inicial

Clique em um estado para selecioná-lo (ele fica com borda laranja). Depois clique em ⭐ Inicial. Uma setinha aparecerá apontando para ele.

3. Definir estados finais

Selecione um estado e clique em 🔘 Alternar Final. O círculo ganha um contorno duplo. Se quiser tirar, clique novamente.

4. Criar transições (as setas)

Nos campos abaixo de "Transições", escolha um estado de origem e um de destino. Digite um símbolo (letra, número ou até ε) e clique em Adicionar. Uma seta aparecerá no desenho.

Importante: Se você criar mais de uma transição com o mesmo símbolo saindo do mesmo estado, o autômato vira não determinístico (AFND). O simulador detecta isso automaticamente e muda a simulação.

💡 Dica: para transições espontâneas (ε), use o botão ε (ε). Elas não consomem símbolo.

▶️ Testando uma palavra

No painel "Simulação Visual", digite uma palavra (por exemplo, "aba") no campo e clique em Executar Automático. Você verá o autômato percorrendo os estados e, no final, será informado se a palavra foi ACEITA ou REJEITADA.

Se você quiser ver passo a passo, clique em Iniciar Passo a Passo. A cada clique em Próximo Símbolo, a máquina lê um caractere e mostra onde está. É ótimo para entender o funcionamento.

🌐 Mudando o idioma

No canto superior direito, há um seletor de idioma. Escolha entre Português, Inglês, Espanhol ou Chinês Simplificado. O simulador se adapta automaticamente.

❓ Perguntas frequentes

Posso arrastar os estados?
Sim! Clique em um círculo e arraste. Eles ficam onde você colocar, facilitando a visualização.

O que significa o símbolo ε?
É uma transição que não consome nenhum símbolo. É como se o autômato "pulase" para outro estado sem ler nada.

Como apagar uma transição?
Na lista de transições, ao lado de cada uma há um botão com ❌. Clique nele para remover.

Como apagar um estado?
Selecione o estado e clique em 🗑️ Remover. Todas as transições que saem ou chegam nele também serão apagadas.

Por que meu autômato ficou "Não Determinístico"?
Se você criou duas transições diferentes com o mesmo símbolo saindo do mesmo estado, ou usou transições ε, ele se torna não determinístico. O simulador muda automaticamente a forma de simular.

📚 Exemplo prático

Objetivo: Criar um autômato que aceite palavras que começam com "a" e terminam com "b".

  1. Crie dois estados: q0 e q1.
  2. Defina q0 como inicial.
  3. Defina q1 como final.
  4. Crie transições:
    • De q0 com "a" vai para q1
    • De q1 com "b" vai para q1
    • De q0 com "b" vai para q0 (para "matar" palavras que começam com b)
  5. Teste a palavra "ab" → deve ser aceita.
  6. Teste "aab" → deve ser aceita.
  7. Teste "ba" → deve ser rejeitada.

Você pode construir isso facilmente no simulador e ver o resultado.

✨ Dicas finais

  • Use o botão 🔄 Organizar para distribuir os estados automaticamente em círculo.
  • Se quiser limpar todas as transições de uma vez, use o botão Limpar todas.
  • O modo passo a passo é ótimo para entender o caminho percorrido.
