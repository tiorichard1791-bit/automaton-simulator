# Simulador de Autômato Finito (AFD/AFND)

Este é um simulador visual de Autômatos Finitos Determinísticos (AFD) e Não Determinísticos (AFND), desenvolvido em HTML, CSS e JavaScript puro, utilizando SVG para a renderização gráfica.

## Funcionalidades

- Criação e edição de estados (arrastáveis, renomeação, inicial/final).
- Criação de transições com símbolos quaisquer, incluindo ε (transição espontânea).
- Detecção automática se o autômato é determinístico ou não.
- Simulação visual automática da cadeia de entrada, destacando estados e transições.
- Modo passo a passo para acompanhamento detalhado da execução.
- Interface amigável e responsiva.
